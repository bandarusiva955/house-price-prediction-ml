# house-price-prediction-ml
Machine Learning project to predict house prices using regression techniques with Python and Scikit-learn.

# House Price Prediction using Machine Learning

## Project Overview
This project predicts housing prices using machine learning regression models. 
The goal is to analyze housing features and build a model that estimates property prices accurately.

## Dataset
Housing dataset containing features such as:

- Median income
- House age
- Number of rooms
- Population
- Location features

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation

## Machine Learning Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- Mean Absolute Error
- Mean Squared Error
- R² Score

## Results
The trained model successfully predicts house prices based on housing attributes.
